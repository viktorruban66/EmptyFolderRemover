EmptyFolderRemover – Удалятор пустых папок на 7 языках
Интеллектуальный инструмент для очистки файловой системы от пустых директорий.
Поддерживает рекурсивный обход, симуляцию, логирование, исключения, удаление папок с «мусорными» файлами и многое другое.
Реализован на семи языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Обнаружение пустых папок – удаляет директории, не содержащие ни одного файла или подпапки.

Рекурсивный режим – проход по всем вложенным папкам (по умолчанию).

Гибкая настройка:

--dry-run – только показать, что будет удалено.

--verbose – подробный вывод.

--log <file> – сохранить лог операций.

--exclude <pattern> – исключить папки, соответствующие шаблону.

--ignore-files <files> – считать папку пустой, если она содержит только указанные файлы (например, .DS_Store, Thumbs.db).

--min-size <bytes> – удалять папку, если она пуста ИЛИ содержит только файлы меньше указанного размера.

Безопасность – режим симуляции и подтверждение перед удалением (опционально).

Цветной вывод – наглядные сообщения об успехе/ошибке.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [путь] [опции]
Основные опции
Опция	Описание
-p, --path <dir>	Корневая папка для сканирования (по умолчанию текущая)
-r, --recursive	Рекурсивно обходить подпапки (включено по умолчанию, можно отключить)
-n, --dry-run	Только показать, что будет удалено, без фактического удаления
-v, --verbose	Подробный вывод (показывает каждую проверенную папку)
-l, --log <file>	Сохранить список удалённых папок в файл
-e, --exclude <pattern>	Исключить папки, соответствующие glob-шаблону (например, *cache*)
-i, --ignore-files <list>	Считать папку пустой, если содержит только перечисленные файлы (через запятую)
-m, --min-size <bytes>	Удалять папку, если она пуста ИЛИ все файлы в ней меньше указанного размера
-y, --yes	Автоматически согласиться на удаление (без запроса)
Примеры
bash
# Удалить все пустые папки в ~/Downloads (рекурсивно)
python remover.py -p ~/Downloads

# Только показать, что будет удалено
python remover.py -n

# Игнорировать папки с "temp" в имени и удалять пустые, содержащие только .DS_Store
python remover.py -e "*temp*" -i ".DS_Store"

# Удалять пустые папки, а также папки, где все файлы меньше 1 КБ
python remover.py -m 1024

# Записать лог и выполнить без подтверждения
python remover.py -y -l removed.log

🧠 Логика работы
Обход директорий (рекурсивно или нет) с использованием os.walk, walk или аналогов.

Для каждой папки проверяется:

Содержит ли она подпапки (если да, она не пуста).

Содержит ли она файлы (кроме игнорируемых).

Все ли файлы имеют размер меньше --min-size (если указано).

Если папка признана «пустой» (или подлежит удалению), она добавляется в список на удаление.

При реальном удалении порядок важен: сначала удаляются вложенные пустые папки, затем родительские (для этого используется обход снизу вверх).

Если включена симуляция – только вывод.

Опционально – запрос подтверждения перед удалением (если не указано --yes).

✨ Дополнительные фичи
Безопасный порядок удаления – сначала самые глубокие папки, чтобы избежать ошибок.

Игнорирование системных файлов – по умолчанию Thumbs.db, .DS_Store, desktop.ini.

Поддержка glob-исключений – можно исключить папки по имени.

Цветной вывод – удалённые папки выделяются зелёным, ошибки – красным.
